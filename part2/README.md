## 🚀 Step-by-Step Hands-on Sandbox Guide

### Step 1: Extract the Semantic Chains (Appendix A & B)
1. Load the graph extraction prompt in **Appendix A** into your LLM.
2. Feed it with the 14 atomic components to see how the AI maps relationships using formalized predicates (`iirds:has-tool`, `iirds:has-supply`).
3. Verify the deterministic result against the verified triplet map in **Appendix B**.

### Step 2: Relive the V4 Atomization Process (Appendix C & D)
1. Run **Appendix C** against the source manual text. This prompt acts as a certified iiRDS auditor, forcing the AI to validate its metadata selections against a hardcoded knowledge base (indicators, `what_it_is_not`).
2. The exact execution result is archived as production-ready JSON metadata in **Appendix D**. Review the `Reasoning` keys to inspect the logical trail of the V4 architecture.

### Step 3: Run the Stress Tests (Appendix E to J)
To fully appreciate the superiority of the Graph-Aware Atomic Architecture over pure vector similarity, replicate the three article experiments:

*   **Replicate TEST A (The Threat of Goodwill):** Ask both systems what to do if the standard SP-X100 filter is stuck. Observe how V3 (**Appendix E**) hallucinates a dangerous cross-model suggestion (recommending the SP-X200 ceramic spanner) out of semantic proximity, whereas V4 (**Appendix F**) exhibits the robust **Discipline of Silence**.
*   **Replicate TEST B (The Logical Rails):** Enforce the hybrid graph override constraints. Witness how both systems (V3 in **Appendix G**, V4 in **Appendix H**) successfully leverage the graph to reject the incorrect tool, proving that an un-connected edge functions as an absolute safety interlock.
*   **Replicate TEST C (The Impedance Mismatch):** Challenge the AI to extract crossing attributes ("pull consumables required strictly for safety during SP-X200 replacement"). Notice how V3 (**Appendix I**) crashes and outputs a blank response because its physical chunks are structurally fused. Conversely, V4 (**Appendix J**) traverses the orthogonal nodes seamlessly to immediately deliver `Chemical-resistant gloves` and `Protective goggles`.
